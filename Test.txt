🧈 ButterLab – הוראות הרצה לפרויקט
דרישות מקדימות

לפני שמריצים את הפרויקט, ודאו שמותקנים:

Node.js (כולל npm)

MySQL Server

MySQL Command Line Client או MySQL Workbench

Git

1️⃣ הורדת הפרויקט מ־GitHub
git clone <REPO_URL>
cd <PROJECT_FOLDER>

2️⃣ התקנת חבילות צד־שרת

להיכנס לתיקיית השרת:

cd ServerSide
npm install

3️⃣ הקמת בסיס הנתונים (בהעתק–הדבק)
3.1 יצירת בסיס הנתונים והטבלאות (SCHEMA)

פתחו MySQL Command Line Client או MySQL Workbench

התחברו כ־root

פתחו את הקובץ schema.sql

העתיקו את כל התוכן

הדביקו ב־MySQL והריצו

בדיקה:

USE butterlab;
SHOW TABLES;


אמורות להופיע הטבלאות:

contacts
products
orders
order_items

3.2 הכנסת נתוני מוצרים (SEED)

פתחו את הקובץ seed.sql

העתיקו את כל התוכן

הדביקו ב־MySQL והריצו

בדיקה:

USE butterlab;
SELECT * FROM products;

4️⃣ עדכון קובץ הקונפיגורציה של בסיס הנתונים

פתחו את הקובץ:

ServerSide/db.config.js


ועדכנו רק את סיסמת ה־MySQL לפי המחשב שלכם:

module.exports = {
  HOST: "localhost",
  USER: "root",
  PASSWORD: "הסיסמה_שלכם_ל־MySQL",
  DB: "butterlab"
};

5️⃣ הרצת השרת

מתוך תיקיית ServerSide:

npm run dev


או:

node Index.js


אם הכל תקין, תופיע ההודעה:

Server listening on http://localhost:3000

6️⃣ בדיקות תקינות
בדיקת חיבור לבסיס הנתונים

בדפדפן:

http://localhost:3000/db-test


אם מתקבל JSON ללא שגיאה — החיבור תקין.

בדיקת מוצרים מהשרת
http://localhost:3000/products


אמורה לחזור רשימת מוצרים מה־DB.

פתיחת האתר
http://localhost:3000

7️⃣ בדיקת הזמנות (מה שמומש בפרויקט)

היכנסו לאתר

הוסיפו מוצרים לעגלה

עברו ל־Checkout ושלחו הזמנה

בדיקה ב־MySQL:

USE butterlab;
SELECT * FROM orders ORDER BY id DESC;
SELECT * FROM order_items ORDER BY order_id DESC;


בדיקה משולבת:

SELECT
  o.id AS order_id,
  o.full_name,
  p.name AS product_name,
  oi.quantity,
  oi.unit_price
FROM orders o
JOIN order_items oi ON oi.order_id = o.id
JOIN products p ON p.id = oi.product_id
ORDER BY o.id DESC;

8️⃣ תקלות נפוצות

DB error – לרוב סיסמה שגויה ב־db.config.js

אין מוצרים – ה־seed.sql לא הורץ

אין פריטים ב־order_items – ה־schema לא עודכן
